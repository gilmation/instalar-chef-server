### Lanza el script de la instalación:

    $ pwd
    instalar-chef-server
    $ ./lanzar
    Usage: lanzar user host

### Configuración

Comprobar la configuración en /etc/chef/*.rb

### Crea un repositorio local

Se puede encontrar un repositorio vacio en:

    $ pwd
    instalar-chef-server
    $ cd chef-repo
